Kino Stream GitHub Pages repozitář

1) Nahrajte OBSAH této složky do veřejného GitHub repozitáře s názvem kinostream-repo.
2) V GitHub Pages nastavte Source = Deploy from a branch, Branch = main, Folder = /(root).
3) Do Kodi přidejte zdroj: https://nxt124.github.io/kinostream-repo/
4) Z něj nainstalujte repository.kinostream-1.1.0.zip
5) Pak v Kodi otevřete Instalovat z repozitáře -> Kino Stream Repository.
